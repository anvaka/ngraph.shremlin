Vertices to Vertices pipe

This pipe take a vertex as an input, and emits adjacent vertices (neighbors) as an output.

module.exports = VerticesToVerticesPipe;

See also: BasePipe

var BasePipe = require('../basePipe'), createFilter = require('../utils/filterExpression'), createEmptyIterator = require('../iterators/emptyIterator'), createEdgeIterator = require('../iterators/edgeIterator'), util = require('util'); util.inherits(VerticesToVerticesPipe, BasePipe);

Depending on mode emitted vertices are:

  • out - emits vertices connected with outgoing edge (out adjacent)
  • in - emits vertices connected with incoming edge (in adjacent)
  • both - emits both in and out adjacent vertices

See Syntactic Sugar for more examples;

Results can be filtered based on edge attributes with filter attribute. See Filtering Pipes.
